# Formity + React Hook Form

Example repository showing how to integrate [Formity](https://formity.app) with [React Hook Form](https://react-hook-form.com/).

## Overview

This repository demonstrates how Formity and React Hook Form work together. Formity handles the multi-step flow logic — branching, looping, jumping — while React Hook Form handles validation and field state within each step.

## Getting started

Clone the repository.

```bash
git clone https://github.com/martiserra99/formity-react-hook-form
```

Install the dependencies.

```bash
npm install
```

Start the development server.

```bash
npm run dev
```
