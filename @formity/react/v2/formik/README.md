# Formity + Formik

Example repository showing how to integrate [Formity](https://formity.app) with [Formik](https://formik.org/).

## Overview

This repository demonstrates how Formity and Formik work together. Formity handles the multi-step flow logic — branching, looping, jumping — while Formik handles validation and field state within each step.

## Getting started

Clone the repository.

```bash
git clone https://github.com/martiserra99/formity-formik
```

Install the dependencies.

```bash
npm install
```

Start the development server.

```bash
npm run dev
```
